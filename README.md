# ☀️ Vikram Solar — AI-Powered Solar Platform

India's most advanced AI solar platform built with **React + Vite**.

## 🚀 Quick Start

### Prerequisites
- Node.js v18+ → https://nodejs.org
- VS Code → https://code.visualstudio.com

### 1. Open in VS Code
```
File → Open Folder → select vikram-solar folder
```

### 2. Open terminal in VS Code (Ctrl + `)
```bash
npm install
npm run dev
```
Opens at http://localhost:5173

### 3. Build for production
```bash
npm run build
```
